# UtangPH Backend

Express.js backend API for the UtangPH expense tracking application.

## Installation

```bash
npm install
```

## Environment Variables

Create a `.env` file in this directory:

```
MONGODB_URI=your_mongodb_connection_string
PORT=5000
```

## Running

Development mode with auto-reload:
```bash
npm run dev
```

Production mode:
```bash
npm start
```
